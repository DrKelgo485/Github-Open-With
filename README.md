# Github-Open-With
Chrome extension that helps you open Github repositories in a different 3rd party IDES.
 When navigating to any github repository, you can click on the green code button and see new options:
- Open in Github Dev
- Open in Github1s
- Open in StackBlitz
- Open in CodeSandbox
